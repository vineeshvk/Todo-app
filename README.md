### Requirements, install as needed:

##### React Native:
```
$ npm i -g react-native-cli
```

##### watchman:

```
$ brew install watchman
```

##### clone this repo:

```
$ cd react-native-todo
$ npm install
$ react-native run-android
```

>Open the directory in the text editor of your choice.
